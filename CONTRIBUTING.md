# Contributing

Thank you for your interest in contributing to **MakerWorld to Snapmaker U1**!

Contributions of all kinds are welcome, including bug reports, feature suggestions, documentation improvements and pull requests.

## Before You Start

For larger changes, please open an Issue first to discuss the proposed implementation. This helps avoid duplicated work and ensures new features align with the project's goals.

## Reporting Bugs

When reporting a bug, please include as much information as possible:

* Browser and version
* Extension version
* Operating system
* MakerWorld model URL (if possible)
* Steps to reproduce the issue
* Expected behavior
* Actual behavior
* Console output (if available)

## Pull Requests

Please keep pull requests focused on a single topic.

Before submitting:

* Test the extension with multiple MakerWorld models whenever possible.
* Keep changes consistent with the existing code style.
* Update documentation if necessary.
* Avoid unrelated refactoring.

## Project Goals

The primary goals of this project are:

* Preserve as much original project information as possible.
* Generate native Snapmaker U1 compatible project files.
* Keep all conversion local inside the browser.
* Maintain compatibility with current MakerWorld downloads.
* Keep the converter simple, reliable and easy to maintain.

Thank you for helping improve the project!
